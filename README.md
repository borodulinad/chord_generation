В данном репозитории представлен проект в рамках курса "Анализ текстов. Генеративные модели".

В качестве датасета взят https://huggingface.co/datasets/lluccardoner/melodyGPT-song-chords-text-1

В рамках проекта сделано следующее:

1. Обработка данных
2. Аугментация через транспонирование аккордов
3. Реализация токенайзера аккордов.
4. Обучение модели BertLM
5. Файнтюннинг на жанр.
7. Построен график с эмбеддингами песен с помощью PCA.
7. Написан сервис через gradio, в котором можно интерактивно вводить аккорды и получать предсказание на замаскированный аккорд, а также прослушать получившуюся композицию.

## Материалы

1) Ноутбук train_bert.ipynb - подготовка данных, реализация токенайзера, обучение BertLM, файнтюн на жанр, отрисовка эмбеддингов PCA
2) Ноутбук bert_predict.ipynb - реализация функции predict, а также поднятие сервиса через gradio.
3) Модель best_genre_bert.pth - модель BertLM с файнтюном на жанр.
4) Токенайзер chord_tokenizer.pth - обученный токенайзер.
5) Класс chord_tokenaizer.py - для удобства вынесли в py-файл, чтобы не тащить в ноутбук с предиктом
6) Класс bert_architecture.py - для удобства архитектуру вынесли в py-файл, чтобы не тащить в ноутбук с предиктом.
